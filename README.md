# DevOps Project 

This repository contains my DevOps practice labs to create a basic online bookstore integrated k8s deployment and a CI/CD pipeline.

## Technologies

- Visual Studio Code
- Git
- GitHub repositories
- GitHub Actions and workflow
- Docker
- Docker Hub
- Minikube
- Kubernetes


## K8s Deployments

- Nginx Deployment
- flask-API Deployment
- redis Deployment
- NodePort and load Balancer Service
- ConfigMap Lab
- Secrets
- Volumes
- Persistent Volumes and PVC
- Storageclass
